# 測試檢查清單

使用此清單來驗證原型的各項功能是否正常運作。

## 安裝測試

- [ ] 成功產生三個圖示檔案（icon16.png, icon48.png, icon128.png）
- [ ] Chrome擴充功能頁面顯示「YouTube字幕電子書製作器」
- [ ] 沒有錯誤訊息
- [ ] 擴充功能圖示出現在工具列

## 基本功能測試

### 1. Popup介面
- [ ] 點擊擴充功能圖示，彈出視窗正常顯示
- [ ] 可以選擇不同的截圖間隔（1秒、2秒、3秒、5秒）
- [ ] 可以選擇不同的每頁行數（3、5、7、10行）
- [ ] 可以輸入字幕區域高度數值

### 2. 在非YouTube頁面
- [ ] 在非YouTube頁面點擊「開始製作電子書」
- [ ] 應該顯示錯誤訊息：「請在YouTube影片頁面使用此功能」

### 3. 在YouTube影片頁面
- [ ] 開啟任何YouTube影片
- [ ] 點擊擴充功能圖示
- [ ] Popup正常顯示
- [ ] 點擊「開始製作電子書」
- [ ] 影片自動暫停
- [ ] 顯示「正在初始化...」訊息
- [ ] 進度條出現並開始增長

## 截圖功能測試

- [ ] 影片會自動跳轉到不同時間點
- [ ] 可以看到影片在逐格截圖
- [ ] 進度訊息顯示「截圖中... X/Y」
- [ ] 進度條從0%增長到50%（截圖階段）
- [ ] 沒有出現錯誤訊息

## 組合功能測試

- [ ] 截圖完成後，顯示「組合頁面中...」
- [ ] 進度訊息顯示「組合頁面... X」
- [ ] 進度條從50%增長到100%
- [ ] 顯示完成訊息，例如：「完成！已截取 300 張圖片，共 60 頁」
- [ ] 「開啟閱讀器」按鈕出現

## 閱讀器測試

### 基本顯示
- [ ] 點擊「開啟閱讀器」
- [ ] 全螢幕黑色覆蓋層出現
- [ ] 顯示影片標題
- [ ] 顯示頁數（例如：1 / 60）
- [ ] 顯示多行字幕截圖
- [ ] 每行字幕旁邊顯示時間戳記

### 導航功能
- [ ] 「上一頁」按鈕在第一頁時為disabled狀態
- [ ] 點擊「下一頁」可以切換到下一頁
- [ ] 頁數正確更新
- [ ] 字幕內容正確切換
- [ ] 點擊「上一頁」可以回到上一頁
- [ ] 「下一頁」按鈕在最後一頁時為disabled狀態

### 音訊播放
- [ ] 點擊「播放」按鈕
- [ ] YouTube影片開始播放音訊
- [ ] 播放的音訊對應當前頁面的時間範圍
- [ ] 播放到該頁結束時間時自動暫停

### 關閉功能
- [ ] 點擊右上角的「✕」按鈕
- [ ] 閱讀器關閉
- [ ] 回到YouTube頁面
- [ ] 影片仍在原位置

## 視覺測試

### Popup樣式
- [ ] 所有文字清晰可讀
- [ ] 按鈕有hover效果
- [ ] 進度條顏色正確（綠色）
- [ ] 訊息區域背景色正確（灰色）

### 閱讀器樣式
- [ ] 背景為深色（黑色/深灰）
- [ ] 字幕圖片清晰
- [ ] 時間戳記右對齊
- [ ] 按鈕有hover效果
- [ ] 播放按鈕為紅色
- [ ] 捲動條樣式正確

## 效能測試

### 短影片（< 5分鐘）
- [ ] 截圖速度可接受（每秒1-2張）
- [ ] 不會當機或無回應
- [ ] 記憶體使用量正常

### 中等影片（5-15分鐘）
- [ ] 可以完整處理
- [ ] 進度正確顯示
- [ ] 最終可以開啟閱讀器

### 注意事項
如果測試長影片（>15分鐘），可能需要：
- [ ] 調大截圖間隔（3-5秒）
- [ ] 確認瀏覽器有足夠記憶體

## 邊界情況測試

- [ ] 影片長度少於10秒時仍可運作
- [ ] 沒有字幕的影片也能截圖（只是會是空白）
- [ ] 切換不同影片後重新製作電子書
- [ ] 關閉閱讀器後再次開啟，資料仍在

## 錯誤處理測試

- [ ] 在製作過程中重新整理頁面，不會出現錯誤
- [ ] 在截圖過程中切換YouTube影片，程式正確中斷或提示
- [ ] 開啟開發者工具（F12）檢查Console，無嚴重錯誤

## 已知限制確認

確認以下限制在文件中有說明：
- [ ] 僅適用硬編碼字幕
- [ ] 長影片會佔用大量記憶體
- [ ] 製作過程需要時間
- [ ] 字幕區域需要手動調整
