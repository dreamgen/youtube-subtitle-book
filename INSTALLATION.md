# 安裝說明

## 前置需求

- Google Chrome瀏覽器（版本90以上）
- 本專案資料夾

## 安裝步驟

### 方法一：使用圖示產生工具（推薦）

1. **產生圖示檔案**
   ```bash
   # 用瀏覽器開啟圖示產生器
   open create-icons.html
   # 或在Windows上
   start create-icons.html
   ```

2. 點擊「產生圖示檔案」按鈕

3. 會自動下載三個檔案，將它們放到專案資料夾中：
   - icon16.png
   - icon48.png
   - icon128.png

4. **載入擴充功能**
   - 開啟Chrome
   - 輸入網址：`chrome://extensions/`
   - 開啟「開發人員模式」（右上角開關）
   - 點擊「載入未封裝項目」
   - 選擇本專案資料夾：`youtube-subtitle-book`

5. **完成！**
   - 擴充功能會出現在列表中
   - 工具列會出現紅色「YT」圖示

### 方法二：手動建立圖示

如果無法使用圖示產生工具，可以手動建立：

1. **使用任何圖片編輯軟體**（如PhotoShop、GIMP、Paint等）

2. **建立三個PNG圖片**：
   - icon16.png（16x16像素）
   - icon48.png（48x48像素）
   - icon128.png（128x128像素）

3. 圖片內容任意（建議使用紅色背景 + 白色「YT」文字）

4. 將三個圖片放到專案資料夾中

5. 繼續上述「載入擴充功能」步驟

### 方法三：使用線上工具

1. 前往任何線上圖示產生器，例如：
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. 上傳任意圖片或建立簡單圖示

3. 下載產生的PNG檔案

4. 重新命名為：icon16.png、icon48.png、icon128.png

5. 放到專案資料夾中

## 驗證安裝

### 檢查清單

- [ ] `chrome://extensions/` 頁面顯示「YouTube字幕電子書製作器」
- [ ] 沒有紅色錯誤訊息
- [ ] 擴充功能狀態為「已啟用」
- [ ] 工具列出現擴充功能圖示

### 快速測試

1. 開啟任何YouTube影片

2. 點擊擴充功能圖示

3. 應該看到控制面板彈出視窗

4. 如果在非YouTube頁面，點擊「開始製作」會顯示錯誤訊息

## 疑難排解

### 問題：找不到擴充功能圖示

**解決方法**：
1. 點擊Chrome右上角的拼圖圖示（擴充功能選單）
2. 找到「YouTube字幕電子書製作器」
3. 點擊釘選圖示（📌）將它固定在工具列上

### 問題：載入擴充功能時出現錯誤

**可能原因及解決方法**：

1. **缺少圖示檔案**
   ```
   錯誤：Could not load icon 'icon16.png'
   解決：確認三個圖示檔案都在資料夾中
   ```

2. **manifest.json格式錯誤**
   ```
   錯誤：Manifest file is invalid
   解決：檢查manifest.json是否被修改，確認JSON格式正確
   ```

3. **資料夾路徑錯誤**
   ```
   錯誤：無法載入
   解決：確認選擇的是專案根目錄（包含manifest.json的資料夾）
   ```

### 問題：擴充功能無法在YouTube上運作

**檢查步驟**：

1. **確認權限設定**
   - 進入 `chrome://extensions/`
   - 找到本擴充功能
   - 點擊「詳細資料」
   - 確認「網站存取權」為「在特定網站上」
   - 應該包含 `youtube.com`

2. **重新載入擴充功能**
   - 在擴充功能頁面點擊「重新載入」圖示
   - 重新整理YouTube頁面

3. **檢查Console錯誤**
   - 在YouTube頁面按F12開啟開發者工具
   - 切換到「Console」標籤
   - 查看是否有紅色錯誤訊息

### 問題：開發者模式無法開啟

**可能原因**：
- 公司或學校電腦可能被管理員限制
- 需要管理員權限

**解決方法**：
- 使用個人電腦
- 或聯絡IT管理員申請權限

## 更新擴充功能

當修改程式碼後：

1. 進入 `chrome://extensions/`

2. 找到「YouTube字幕電子書製作器」

3. 點擊「重新載入」圖示（🔄）

4. 重新整理使用中的YouTube頁面

## 解除安裝

1. 進入 `chrome://extensions/`

2. 找到「YouTube字幕電子書製作器」

3. 點擊「移除」按鈕

4. 確認移除

**注意**：移除擴充功能會清除所有儲存的電子書資料
