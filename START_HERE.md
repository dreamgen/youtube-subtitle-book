# 🌟 從這裡開始！

歡迎使用 **YouTube 字幕電子書製作器 V2.0**！

這是一個 Chrome 擴充功能，可以將 YouTube 影片的硬編碼字幕截圖，組合成可翻頁閱讀的電子書格式。

## ✨ V2.0 新亮點

| 功能 | 說明 |
|------|------|
| 🎨 **Side Panel** | 全新側邊面板介面，操作更便利 |
| 📚 **書庫管理** | 統一管理所有儲存的字幕段落 |
| 📄 **PDF 導出** | 單檔或合併導出 PDF |
| 📖 **有聲書導出** | HTML 格式可離線播放 |
| 🤖 **智慧擷取** | 偵測字幕變化自動截圖 |
| 📖 **即時閱讀** | 製作過程中即時預覽 |

---

## ⚡ 5 分鐘快速開始

### 步驟 1: 安裝擴充功能

1. 開啟 Chrome，進入 `chrome://extensions/`
2. 開啟右上角「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇這個資料夾 `youtube-subtitle-book-v2`

### 步驟 2: 開始使用

1. 前往 YouTube 影片頁面
2. 點擊擴充功能圖示，開啟側邊面板
3. 設定參數（或使用預設值）
4. 點擊「🚀 開始製作」

### 步驟 3: 閱讀與管理

- **即時閱讀**：製作中點擊「📖 即時閱讀」
- **完成後閱讀**：點擊「✅ 開啟閱讀器」
- **書庫管理**：點擊「📂 開啟書庫管理」

## 💡 使用情境

### 情境 1：學習外語
```
1. 找一個有硬編碼字幕的外語教學影片
2. 使用智慧偵測模式擷取字幕
3. 開啟閱讀器，一邊看字幕一邊播放音訊
4. 儲存到書庫，隨時復習
```

### 情境 2：製作筆記
```
1. 擷取演講或教學影片的字幕
2. 在閱讀器中編輯（刪除不需要的部分）
3. 導出為 PDF 作為學習筆記
```

### 情境 3：離線閱讀
```
1. 製作完成後儲存到書庫
2. 使用「HTML 有聲書導出」
3. 將 HTML 檔案保存，可離線開啟
```

## ⚠️ 重要提醒

1. **僅支援硬編碼字幕**
   - ✅ 影片中嵌入的字幕（burned-in）
   - ❌ YouTube 自動字幕 (CC)

2. **調整字幕區域**
   - 不同影片的字幕位置可能不同
   - 使用「預覽」功能確認效果
   - 調整「字幕區域高度」和「底部邊界」

3. **長影片處理時間**
   - 10分鐘影片約需 5-10 分鐘處理
   - 可使用「製作範圍」限制頁數
   - 建議先製作小範圍測試
