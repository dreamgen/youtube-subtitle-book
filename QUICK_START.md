# ⚡ 快速開始 - 5分鐘上手！

## 📦 安裝擴充功能

### 步驟 1: 開啟擴充功能頁面
在 Chrome 網址列輸入：
```
chrome://extensions/
```

### 步驟 2: 開啟開發人員模式
點擊右上角的「開發人員模式」開關

### 步驟 3: 載入擴充功能
1. 點擊「載入未封裝項目」
2. 選擇 `youtube-subtitle-book-v2` 資料夾
3. 完成！

---

## 🎬 第一次使用

### 步驟 1: 開啟 YouTube 影片
前往任何有**硬編碼字幕**的 YouTube 影片
（注意：不是 YouTube 自動字幕）

### 步驟 2: 開啟側邊面板
1. 點擊瀏覽器工具列的擴充功能圖示
2. 側邊面板會自動開啟

### 步驟 3: 調整參數（可選）
- **每頁行數**：5（預設）
- **製作範圍**：建議先選「預覽 5 頁」測試
- **開始時間**：從當前時間

### 步驟 4: 預覽效果（推薦）
1. 展開「進階設定」
2. 點擊「🔍 預覽效果」
3. 查看截圖位置是否正確
4. 如不正確，調整「字幕區域高度」

### 步驟 5: 開始製作
點擊「🚀 開始製作」按鈕

等待製作完成...

### 步驟 6: 開啟閱讀器
製作完成後，點擊「✅ 開啟閱讀器」

---

## 📖 閱讀器操作

| 按鈕 | 功能 |
|------|------|
| ◀ 上一頁 | 前往上一頁 |
| ▶ 下一頁 | 前往下一頁（自動播放） |
| ▶ 播放 | 播放當前頁音訊 |
| 🔇 靜音 | 開關靜音 |
| 1.0x | 調整播放速度 |

### 編輯功能
- **➕ 插入**：在下方插入新截圖
- **🗑 刪除**：刪除當前截圖
- **⏱ 時間**：調整截圖時間
- **📍 上方**：擷取上方字幕

---

## 📚 儲存與管理

### 儲存段落
1. 製作完成後會自動儲存
2. 可在側邊面板的「最近儲存」中看到

### 書庫管理
1. 點擊「📂 開啟書庫管理」
2. 可以：
   - 載入已儲存的段落
   - 刪除不需要的段落
   - 批次選取多個段落
   - 導出為 PDF 或有聲書

---

## 📄 導出功能

### PDF 導出
1. 開啟書庫管理
2. 選取要導出的段落
3. 點擊「PDF 導出」

### HTML 有聲書
1. 開啟書庫管理
2. 選取段落
3. 點擊「有聲書導出」
4. 下載的 HTML 可離線開啟

### 資料備份
1. 點擊「匯出全部」
2. 下載 JSON 檔案
3. 需要時可「匯入」還原

## 🤖 智慧擷取模式（進階）

比固定間隔更精準！

### 開啟方式
1. 展開「進階設定」
2. 擷取模式選擇「🤖 智慧偵測 (Beta)」

### 原理
- 偵測到字幕變化時才截圖
- 減少重複截圖
- 更精準捕捉字幕

### 參數調整
- **靈敏度**：建議 20-40%
- **最小像素比例**：建議 0.5%
- **檢測區域寬度**：建議 15%

---

## ⚠️ 常見問題

### Q: 截圖全黑？
A: 調整「字幕區域高度」，每個影片的字幕位置不同

### Q: 無法使用？
A: 確認是在 YouTube 影片頁面（`youtube.com/watch?v=...`）

### Q: 製作很慢？
A: 正常現象。可使用「製作範圍」限制頁數

### Q: 字幕沒有截到？
A: 使用「預覽效果」確認位置，調整高度和間隔

## 📊 建議設定

### 快速測試
```
每頁行數：5
製作範圍：預覽 5 頁
截圖間隔：2 秒
```

### 一般製作
```
每頁行數：7
製作範圍：全部
截圖間隔：2-3 秒
```

### 長影片
```
每頁行數：10
製作範圍：每次 30 頁
截圖間隔：3-5 秒
```
